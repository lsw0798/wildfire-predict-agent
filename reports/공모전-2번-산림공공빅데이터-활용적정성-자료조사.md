## 기획서 2번에 넣을 수 있는 핵심 문장 초안

본 서비스는 산림청·산림빅데이터 플랫폼의 산불 발생 이력, 전국 산불 발생 위치도, 오늘 산불 API를 결합하여 산불 예측·탐지·의사결정 지원에 필요한 과거성·공간성·실시간성을 동시에 확보한다. `WSQ000301` 산불발생 경향성 데이터는 발생 시점, 위치, 원인, 피해면적, 기상·임상·지형·접근성 변수를 포함하고 있어 입력 좌표 주변의 과거 산불 패턴과 위험요인을 정량화하는 데 적합하다. 또한 `todayFireGet` API는 실시간 산불 상태와 기상성 위험 신호를 반영해 과거 데이터만 사용할 때의 시차 문제를 보완한다. 전국 산불 발생 위치도 Shapefile은 향후 지도 기반 히트맵, 행정구역별 취약지 분석, 공간 클러스터링에 확장 가능하다. 따라서 본 프로젝트의 데이터 구성은 공공·빅데이터의 활용 목적, 서비스 기능, 의사결정 지원 시나리오와 직접 연결되어 있어 활용 적정성이 높다.

## 프로젝트 기능과 데이터 연결 요약

- 과거 이력 분석: `WSQ000301.csv` → 입력 좌표 반경 내 산불 이력 검색, 주요 원인/최근연도/인접 건수 산출.
- 실시간 위험 반영: `todayFireGet` → 실시간 source로 조회, 습도·풍속 등 위험 feature 우선 반영.
- 지도/공간 확장: `산불발생위치도_전국` Shapefile → 전국 발생 위치도, 히트맵, 공간 클러스터링 근거.
- 서비스 신뢰도 관리: historical/realtime source가 live/fallback/skipped인지 기록하고, 결측·추정치 사용 시 confidence 및 uncertainty note에 반영.
- AI 보조/데모 데이터: AI Hub JSON → 사용자 유형, 기상·지형·연료·사회 인프라 feature 구조 검증 및 `incidents.json` 생성.

## 확인 필요 항목

1. `https://www.bigdata-forest.kr/product/WSQ000301`와 `https://www.bigdata-forest.kr/product/FRT000101`의 현재 접속 가능 여부 및 공식 목록명.
2. `todayFireGet`의 공식 API 명칭, 이용신청/인증키 발급 페이지, 전체 응답 스키마.
3. AI Hub 샘플의 정확한 공식 데이터셋명과 공모전 제출물에 표기해야 할 라이선스/출처 문구.
4. `산불발생위치도_전국` Shapefile이 현재 서비스 코드에서 직접 사용되는지 여부. 현재 확인된 핵심 경로는 CSV 기반 historical 조회다.
